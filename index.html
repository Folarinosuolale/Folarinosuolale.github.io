Project 1 - Fraud Detection model

Project 2: Investment guidance model on startupsuccess

Description: This project is focused on performing a multiple linear regression analysis on a dataset of 50 startups, examining the relationship between various factors and their success.

Methodologies used: Multiple linear regression, Backward elimination.

Datasets used: CSV file containing startup data.

Project 3: Stock prediction

Description: Stock price prediction is a widely popular application of machine learning and data science.

Methodologies used: Random Forest Regression model.

Datasets used: CSV file containing historical stock market data.

Project 4: Predicting Salaries based on Years of Experience

Description: This project involves building a simple linear regression model to predict salaries based on years of experience.

Methodologies used: Simple linear regression.

Datasets used: Dataset of salary and experience data.

Project 5: Prediction of User's Purchase Decision

Description: This project provides an implementation of the Support Vector Machine (SVM) algorithm to classify whether a user will purchase a product on a social networking site based on age and estimated salary.

Methodologies used: Support Vector Machine (SVM).

Datasets used: Social_Network_Ads.csv.

Project 6: Salary Prediction for HR

Description: This Python project is an implementation of polynomial regression to predict salaries based on position levels.

Methodologies used: Polynomial regression.

Datasets used: 'Position_Salaries.csv' dataset.

Project 7: Investigating Netflix Movies Duration Collapse

Description: In this project, I implemented powerful data analysis and visualization capabilities of the pandas and matplotlib libraries in Python to delve into trends in Netflix movies.

Datasets used: Python dictionary of movie durations, CSV file containing information on Netflix movies.

Project 8: The Android App Market on Google Play

Description: Google Play Store Apps and Reviews Analysis.

Datasets used: apps.csv and user_reviews.csv files.

Project 9: US Birth Rate Analysis

Description: The US birth rate analysis project is a data science project that aims to explore and analyze the trends and patterns in US birth rates over time.
